# 📚 Books

A Kotlin-based application for managing and exploring a personal book collection.

## 📝 Overview

This project is designed to help users organize and interact with their personal libraries. It offers features such as:

- Adding new books with details like title, author, and genre.
- Searching and filtering through the collection.
- Tracking reading progress and personal notes.

## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:

- [Kotlin](https://kotlinlang.org/) 1.5 or higher
- [Gradle](https://gradle.org/) 7.0 or higher

### Installation

1. **Clone the repository:**

   ```bash
   git clone https://github.com/KristianHans04/Books.git
   cd Books
   ```

2. **Build the project:**

   ```bash
   ./gradlew build
   ```

3. **Run the application:**

   ```bash
   ./gradlew run
   ```

## 🛠️ Project Structure

- `app/`: Contains the main application code.
- `gradle/`: Gradle wrapper files.
- `.gitignore`: Specifies files to be ignored by Git.
- `build.gradle.kts`: Build configuration script.
- `settings.gradle.kts`: Settings for the Gradle build.
